# Mixture of Experts (MoE) - Área de Trabajo

Este es un área de trabajo dedicada al desarrollo, investigación y evaluación de modelos, con un enfoque principal en la arquitectura de **Mixture of Experts (MoE)** (Mezcla de Expertos).

## Estructura del Directorio

A continuación se detalla la estructura principal de las carpetas activas dentro del proyecto:

- **`MoE/`**: Directorio principal que contiene el código fuente, experimentaciones y el desarrollo de los modelos de *Mixture of Experts*.
- **`Data CTF/`**: Carpeta destinada a alojar los conjuntos de datos (datasets), capturas de tráfico (pcap) o datos de retos tipo CTF que se utilizan para el entrenamiento y prueba de los modelos.
- **`Lectura/`**: Espacio reservado para almacenar bibliografía, *papers*, literatura relacionada y material de estudio y referencia necesario para respaldar la investigación sobre modelos MoE y la tesis.

> **Nota:** Existen otras carpetas de gestión interna excluidas expresamente del seguimiento de versiones (como `Repositorio/` y `Tesis/`), a fin de mantener en el repositorio únicamente todo lo pertinente a experimentación y código del MoE.
